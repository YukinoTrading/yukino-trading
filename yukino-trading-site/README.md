# Yukino Trading Website (Next.js)

## Run locally
1) Install Node.js (LTS)
2) In this folder:
   - npm install
   - npm run dev
3) Open http://localhost:3000

## Customize
- Replace sample products in: app/_data/products.ts
- Update WhatsApp/email in: app/contact/page.tsx
- Update SEO title/description in: app/layout.tsx
